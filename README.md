1. Пітч (Pitch)
Вебплатформа для підготовки до НМТ — це сучасний онлайн-інструмент, що допомагає абітурієнтам систематизувати знання та ефективно готуватися до іспитів. Проєкт вирішує проблему обмеженого доступу до якісних тренувальних матеріалів, надаючи користувачам можливість проходити тести та відстежувати прогрес у реальному часі.

2. Ядро цінності (MVP)
Мінімально життєздатний продукт (MVP) зосереджений на одному ключовому сценарії: авторизований учень обирає предмет, проходить один базовий тест і миттєво отримує результат із відображенням кількості правильних відповідей. Це дозволяє перевірити головну гіпотезу — чи зручна система для швидкої самоперевірки.

3. Карта стейкхолдерів (Stakeholders)
Користувачі (Абітурієнти та учні 11 класу): отримують основну цінність — знання та практику.

Викладачі: зацікавлені в інструменті для моніторингу успішності та надання матеріалів.

Адміністратори платформи: відповідають за наповнення бази тестів та технічну підтримку.

Розробник (Кузнецова О.О.): відповідає за реалізацію функціоналу та дотримання методології Agile.

4. Мета та ознаки успіху (Definition of Success)
Мета першої ітерації: Розробити стабільний модуль тестування з автоматичною перевіркою результатів.

Ознаки успіху:

Система безпомилково підраховує бали після завершення тесту.

Статистика пройденого тесту коректно зберігається в профілі користувача.

Інтерфейс дозволяє пройти тест без залучення сторонніх інструкцій чи допомоги.
